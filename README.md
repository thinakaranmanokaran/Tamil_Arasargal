## ğŸ“œ About the Project

**Rulers of Ancient Tamizhagam** is an educational web project created to preserve, present, and celebrate the legacy of ancient Tamil rulers. The project narrates the political, cultural, and moral contributions of the rulers of **Chera, Chola, and Pandya dynasties**, spanning from the **Sangam age to the early medieval period**.

The content is presented in **both Tamil and English**, ensuring authenticity while remaining accessible to a wider audience.
